Мова Програмування
JavaScript

JavaScript – це мова програмування, що дозволяє зробити Web -сторінку інтерактивною, тобто такою що реагує на дії користувача.

Розробка веб-сайтів

JavaScript - використовується, коли веб-сторінку потрібно зробити динамічною та додати спеціальні ефекти на такі сторінки.

Ігри

Поєднання JavaScript та HTML5 робить JavaScript популярним і в розробці ігор. Він надає бібліотеку Ease JS, яка пропонує прості рішення для роботи з насиченою графікою.

Представлення

JavaScript також забезпечує можливість створення презентацій як веб-сайту. JavaScript забезпечує бібліотеки RevealJS та BespokeJS для створення веб-слайдів.

Серверні програми

Node JS побудований на JavaScript під час виконання Javascript для створення швидких та масштабованих мережевих додатків.

Перетворення типів у JavaScript реалізовано по-різному для операторів складання та віднімання: Зверніть увагу, що в деяких випадках, коли масив є одним з операндів оператора віднімання, масив може бути перетворений до числа автоматично.

Порожній блок коду {} в одних випадках може поводитися як null, а в інших - як "нічого", хоча сам по собі інтерпретується як undefined.

JavaScript був створений за 10 днів.

Існує атака CSRF, відома як «JavaScript hijacking», під час якої тег на сайті зловмисника пошкоджує сторінку на сайті жертви, яка повертає конфіденційну інформацію, таку як JavaScript або JSON.

JavaScript має два нуля. Один називається null, інший — undefined.

JavaScript має 100% зворотну сумісність. Це означає, що код JavaScript, написаний у 1990 році, працюватиме без проблем і сьогодні. Це також означає, що код, який працює сьогодні, завжди буде виконуватися в майбутніх версіях JavaScript.
